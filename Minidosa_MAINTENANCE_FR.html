Maintenance préventive

Vidanger les tuyaux

Le tuyau sous le réservoir doit être branché.

Le robinet doit être fermé.

  1. Fermer le robinet sous le réservoir.
  2. Placer un récipient sous le poste de remplissage.
  3. Tourner le sélecteur AUTO / O / MAIN sur la position O.
  4. Appuyer sur le bouton PURGE autant de fois que nécessaire.

Lorsque les tuyaux sont vides, aucun liquide ne rentre dans la seringue du poste de remplissage.

Vidanger le réservoir

Les tuyaux doivent être vidangés, .

Le robinet doit être fermé.

  1. Débrancher le tuyau sous le réservoir.
  2. Placer un récipient sous le robinet du réservoir.
  3. Ouvrir le robinet sous le réservoir pour que le liquide coule dans le récipient.

Nettoyer les tuyaux

Les tuyaux doivent être vidangés, .

  1. Alimenter le réservoir en eau claire.
  2. Vider à nouveau les tuyaux ().

Maintenance curative

Changer l’ampoule du bouton-voyant EN SERVICE

La machine doit être hors tension.

  1. Dévisser le capuchon transparent;
  2. Dévisser le capuchon coloré.
  3. Appuyer et tourner un doigt posé sur l’ampoule pour la dévisser.
  4. Appuyer et tourner un doigt posé sur la nouvelle ampoule pour la visser.
  5. Visser le capuchon vert.
  6. Visser le capuchon transparent.

Un disque de plastique blanc est posé dans le capuchon coloré.

Les capuchons sont fragiles, visser avec précaution.

Changer l’ampoule du voyant DEFAUT

La machine doit être hors tension.

  1. Dévisser le capuchon coloré.
  2. Dévisser l’ampoule.
  3. Visser la nouvelle ampoule.
  4. Visser le capuchon coloré.

Un disque de plastique blanc est posé dans le capuchon coloré.

Les capuchons sont fragiles, visser avec précaution.
